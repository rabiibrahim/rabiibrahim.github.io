# Rabi Ibrahim Portfolio

A modern, high-performance portfolio for Senior Staff Engineer Rabi Ibrahim.

## Core Stack
- **Native ES6+ JavaScript**: No frameworks, zero overhead.
- **Tailwind CSS**: Utility-first styling via CDN.
- **FontAwesome**: Modern iconography.

## Structure
- `index.html`: Main application entry and structure.
- `index.css`: Custom layout and theme styles.
- `index.js`: Business logic and dynamic rendering.

## Deployment
Upload these three files to any static host (like GitHub Pages) to deploy.
